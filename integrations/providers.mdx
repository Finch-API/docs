# Providers

Finch supports integrations with over 200 Human Resource Information System (HRIS) providers, listed below. For each provider you will find:
* Provider display name
* Provider `id` (used in the `payroll_provider` parameter in [Finch Connect](/docs/Implementation-Guide/Connect/Set-Up-Finch-Connect.md))
* Finch Products supported for the provider (Organization, Payroll, Deductions)
* Integration types available for each supported Product (Automated, Assisted)

**Please note:** Many HRIS providers include payroll functionality, but some do not. If there is no payroll data available from a provider, it will be marked **N/A**. Providers marked <span style={{ color: 'red' }}>Not Supported</span> support payroll data, but an integration is not avaiable due to constraints by the provider. 

### Integration Types
If Finch supports an integration, it will be marked as one of two integration types: <span style={{ color: 'darkgreen' }}>Automated</span> or <span style={{ color: 'goldenrod' }}>Assisted</span>. Automated integrations refresh data every **24 hours**; while assisted integrations refresh data every **7 days** and require a [configuration period](/docs/Integrations/Integration-Types.md#configuration-periods). [Learn more about Integration types](/docs/Integrations/Integration-Types.md).

### How to Request New Integrations
Finch supports Organization data for every provider listed below. 

If you are interested in Payroll or Deductions data for an integration that is labeled "Upon Request" or is not listed, please reach out to developers@tryfinch.com. Our team will assess the request and respond with a potential timeline.
